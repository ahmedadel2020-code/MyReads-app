# MyReads Project

# Demo-Preview
[MyReads-project](https://react-my-reads-app-ahmed.herokuapp.com/)

## Overview
- This app will let you search for books that you are interested in.
- categorized it whether 'currently reading', 'want to read', 'read'.
- change the shelf from any book and it will automatically show on the correct shelf.

## Instruction

* install all project dependencies with `npm install`
* start the development server with `npm start`
